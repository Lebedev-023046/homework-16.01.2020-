Блок навигации

  • Ширина 15 em
  • Должен присутствовать цвет фона
  • Располагается слева
  • Блок навигации наполните контентом, достаточным, чтобы по высоте он начинал захватывать вторую статью, как показанно в данном примере.

Статья 1

Размер блока: Заниает всю доступную ширину Должен присутствовать цвет фона Во всех статьях сделайте отступ снизу от соседей равным .02em при помощи свойства margin-bottom.

Статья 2

Размер блока: Занимает всю доступную ширину Должен присутствовать цвет фона Во всех статьях сделайте отступ снизу от соседей равным .02em при помощи свойства margin-bottom. Картинка вставлена при помощи тега, плавание организованно по правому краю внутри данной статьи для этой картинки.

Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещенным внутрь контентом.
